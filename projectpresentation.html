Security Implementation

🔒

Password Protection

Custom takepassword() function using getch() to mask input with asterisks (*) preventing shoulder surfing.

📝

Input Validation

Password confirmation matching, buffer clearing with fflush(), and proper string handling to prevent overflow.

💾

Data Integrity

Atomic file operations for booking cancellation using temp file strategy - write to temp, then replace original.

// Secure Password Input Implementation void takepassword(char pass[]) { int i = 0; char ch; while(1) { ch = getch(); if(ch == ENTER || ch == TAB) { pass[i] = '\0'; break; } else if(ch == BSKSPC) { if(i > 0) { i--; printf("\b \b"); } } else { pass[i++] = ch; printf("*"); } } }

Persistent Storage System

📁 account.txt

Stores user registration data including encrypted passwords, personal details, and contact information.

FirstName LastName Username Phone Gender DOB Password

📁 flights.txt

Master database of all available flights with pricing, timing, and seat availability information.

SrNo Airline FlightNo Src Dest Price Time Seats

📁 booking.txt

Transaction records linking passengers to specific flights with complete journey details.

FirstName LastName Gender Age Flight Src Dest Price Time
💡 Implementation Note: The system uses fopen() with modes "r", "w", "a" for read, write, and append operations. Booking cancellation implements a safe delete pattern using temporary files to prevent data corruption.
